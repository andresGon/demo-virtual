LESSON 1 > GRAMMAR > OBJECT PRONOUNS
We use object pronouns to refer to the object of a sentence or clause. The object pronouns replace the name of the object, when it would be repetitive to use it. For example, in the sentence “His parents must be really upset with him,” we already know that we are talking about James, so it would be repetitive and unnatural to use his name again. In this case we use “him” to refer to James.
Another example is “I can’t really say anything else about it.” Here, the teacher has already mentioned that he is talking about Fernanda’s problems, so to avoid repetition he uses the object pronoun “it”.
Notice that we often use object pronouns after prepositions like with, for or to.
