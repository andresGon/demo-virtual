LESSON 2 > GRAMMAR > USED TO
WE USE “USED TO” TO DESCRIBE ACTIONS THAT WERE REPEATED VARIOUS TIMES IN THE PAST, BUT WHICH DO NOT CONTINUE IN THE PRESENT. NOTICE THE DIFFERENCE BETWEEN THE FOLLOWING SENTENCES:
Affirmative:
I used to live in an apartment in the center of the city.
Negative:
I didn’t use to save much money.
Question:
Did you use to watch TV?
For the negative and question forms, we use “use to”.
Also, remember that we use “used to” or “use to” with the infinitive of the main verb.
For example: “I used to be very shy.”
